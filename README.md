#  Rental Housing Backend (Django)

Backend-приложение для системы аренды жилья (аналог Airbnb).

##  Стек технологий
- Python / Django / DRF
- MySQL
- Redis
- Celery + Celery Beat
- JWT (SimpleJWT)
- SMTP (Email)
- Docker-ready

##  Основной функционал
- Аутентификация и роли пользователей
- Управление объявлениями
- Фильтрация, сортировка и поиск
- История поисковых запросов
- Бронирование жилья
- Email-уведомления
- Отзывы и рейтинги
- Статистика просмотров объявлений
- Популярные объявления и запросы
- Административная панель

## ⚙ Асинхронные задачи (Celery)
- Отправка email
- Учёт просмотров через Redis
- Периодическая синхронизация данных

##  Поиск
Поиск выполняется по:
- названию
- описанию
- городу
- стране
- улице
- почтовому индексу

##  Популярность
- сортировка по просмотрам
- сортировка по количеству отзывов
- популярные поисковые запросы

Reviews module

Приложение apps.reviews реализует отзывы и рейтинги к объявлениям (Listing). Отзыв привязан к конкретной брони (Booking) через OneToOne связь, поэтому по одной брони можно оставить только один отзыв.

Модель Review содержит: listing, booking, author, rating (1..5), text и created_at. Добавлены индексы для выборок по listing и по author.

API endpoint: GET/POST /listings/<listing_id>/reviews/

GET: список отзывов по объявлению (доступно всем)

POST: создание отзыва (только авторизованным)

При создании отзыва валидируется booking_id:

бронь существует и относится к этому listing

бронь подтверждена

по брони ещё нет отзыва

В админке реализован удобный просмотр отзывов (фильтры, поиск, автокомплит).
